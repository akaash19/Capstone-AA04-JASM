GroupConnect

GroupConnect is a private social network that is focused on maintaing a timeline of shared experiences between both families and friends so that those memories can be relived and experienced once again.

What we do:

GroupConnect is an online web application that brings people together so they can relive and relish those shared experiences that value most to us. Whether you are looking to create a group for your family, friends, or loved ones, GroupConnect is perfect for you. GroupConnect allows users to create customizable groups where they can invite others to join. On these group pages, members of the groups are able to post a wide range of different media including text, audio, images, or video. We aim to create a safe place for groups and families to save protect these precious artifact so that not a single memory will be lost.

GroupConnect prides itself on its high-grade security and privacy features. We understand that these memories and experiences mean a lot to you so we treat it the same way by encrypting all data posted within groups. We want our users to feel like their data and their privacy is protected; and we aim to acheive that everyday.

If you want a safe place to post your valuable memories so that they can be passed on and never lost, use the most trusted and most safe choice, use GroupConnect and relive your experiences.
