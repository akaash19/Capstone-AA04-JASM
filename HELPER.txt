RIGHT AFTER CLONING THE REPO DO:
npm install

Pushing local changes:
make a commit, then
git push origin master

USE A BRANCH IN MOST CASES THOUGH
git checkout -b <branch_name>
make a commit, then
git push origin <branch_name>


Pushing changes to Heroku website:
git remote add heroku https://git.heroku.com/jasm-capstone.git
make a commit, then
git push heroku master

ONLY PUSH TO HEROKU IF YOU'VE PUSHED TO MASTER AND ARE SURE YOU SHOULD PUSH TO HEROKU


Run app locally on http://localhost:3001/ (server and client)
npm run dev
